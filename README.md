### Ubuntu image for Web automation (Python 3.7, Chrome, Firefox).

Python 3 with Chromedriver, for running Selenium web automated tests.

Contains:

- Ubuntu v18.04
- Python 3.7
- Selenium
- Chrome
- Firefox
